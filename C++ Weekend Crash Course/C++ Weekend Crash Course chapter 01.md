- human algorithm
	- algorithm
	  ```
	  algorithm for changing tire
	  1. Raise the car.
	  2. Remove the lug nuts that affix the tire to the car.
	  3. Remove the faulty tire.
	  4. Mount the new tire.
	  5. Install the lug nuts.
	  6. Lower the car.
	  ```
	- the processor
	  ```
	  nouns
	  ---
	  car, tire, nut, jack, wrench
	  verbs
	  ---
	  grab, move, release, turn
	  ```
	- the program
	  ```
	  define the phrase "remove leg nut"
	  1. Grab Wrench
	  2. Move Wrench to leg nut
	  3. Turn Wrench counterclockwise five times
	  4. Remove Wrench from lug nut
	  5. Release Wrench 
	  ```
	- more precise program
	```
	 1. Grab wrench
	 2. Move wrench to lug nut
	 3. While lug nut attached
	 4. {
	 5. Turn wrench counterclockwise
	 6. }
	 7. Remove wrench from lug nut
	 8. Release wrench
	```
- the main take away is
	- computers do exactly what you tell them - no less, no more 
	-  computer processor follows literally a string of commands built from a limited & well defined vocabulary
	-  computer processor are smart enough to make simple decisions
